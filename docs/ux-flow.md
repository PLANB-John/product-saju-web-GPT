# UX Flow (MVP 1차)

## 기본 흐름
1. **홈(`/`)**
   - 서비스 소개 확인
   - `궁합 시작하기` 버튼으로 입력 화면 이동
2. **입력(`/match`)**
   - 현재는 placeholder 구조만 제공
   - 추후 입력 항목이 이 위치에 배치될 예정
3. **결과(`/result/[id]`)**
   - mock 결과(점수/관계 유형/해석) 구조 확인

## 정책/안내 접근
- 홈 및 상단 네비게이션에서 `/policy/privacy`, `/policy/notice`로 이동 가능

## 추후 확장 예정
- `/room`, `/join/[roomCode]`는 공동 확인 기능을 위한 예약 라우트이며,
  현재 단계에서는 "준비중" 안내만 제공한다.
