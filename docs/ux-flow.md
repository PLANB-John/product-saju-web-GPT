# UX Flow (MVP 2차 반영)

## 기본 흐름
1. **홈(`/`)**
   - 서비스 소개 확인
   - `궁합 시작하기` 버튼으로 입력 화면 이동
2. **입력(`/match`)**
   - 사용자 A/B 정보 입력
   - 필수값 및 형식 validation 수행
   - 유효한 경우 결과 페이지로 이동
3. **결과(`/result/[id]`)**
   - mock 결과(점수/관계 유형/해석) 확인
   - 입력한 이름 요약 확인
   - `다시 입력하기`로 `/match` 복귀 가능

## 정책/안내 접근
- 홈 및 상단 네비게이션에서 `/policy/privacy`, `/policy/notice`로 이동 가능

## 추후 확장 예정
- `/room`, `/join/[roomCode]`는 공동 확인 기능을 위한 예약 라우트이며,
  현재 단계에서는 "준비중" 안내만 제공한다.
