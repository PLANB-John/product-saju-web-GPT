# 시각 언어 가이드 (브랜드 차별화 4단계)

## 0) 현재 시각 언어가 일반적으로 느껴지는 이유
- 화면별로 보라/핑크 계열의 톤이 조금씩 달라, 브랜드 공통 인상보다 "페이지 단위 스타일"로 보였다.
- 카드/배지/섹션 타이틀이 기능별로 제각각이라, 정보 위계보다 장식 요소가 먼저 읽히는 구간이 있었다.
- 결과 페이지 점수/유형 강조가 있었지만 홈·입력 페이지와 연결되는 시각 문법이 약해 "같은 브랜드 리포트" 감각이 덜했다.

## 1) 컬러 원칙
- **Main**: Warm Amber-Orange 계열 (`#c95a37` ~ `#ea8354`)을 CTA/핵심 포인트에 집중 사용.
- **Support**: Warm Neutral Peach (`#fff8f4`, `#fff3ec`)로 배경과 정보 블록을 안정화.
- **Text**: 본문은 딥 브라운-그레이 (`#4f4742`, `#5f5752`) 중심으로 대비 확보.
- **Border**: `#f3d8cb`, `#ebc8b9` 계열로 카드/입력/구분선을 통일.
- **Usage Rule**:
  - CTA/포커스/링크는 같은 오렌지 계열 문법 사용
  - 정보성 배지는 muted 톤, 상태 강조는 soft 톤으로 분리

## 2) 카드 원칙
- 공통 반경: `rounded-[1.6rem]`
- 공통 표면: 밝은 warm white + 얇은 peach border + 부드러운 단일 그림자
- 카드 계층:
  1. `surface-card`: 주 콘텐츠 카드
  2. `surface-muted`: 안내/보조 정보 카드
- 홈 /match /result의 입력/결과/안내 카드가 동일 계열로 보이도록 유지

## 3) 배지 원칙
- `badge-soft`: 상단 맥락/핵심 카테고리(예: 저장된 관계 리포트)
- `badge-muted`: 보조 라벨/상태(예: 기준 정보, 리포트 지표)
- 배지는 텍스트 위계를 보조하며, 섹션 헤더와 함께 문서형 리듬을 만든다.

## 4) 강조 원칙
- 점수 강조: 오렌지 단색 기반 ring gauge + 내부 neutral plate 구조
- 관계 유형: soft badge, 보조 지표: muted badge로 분리
- CTA 강조: gradient primary 1종 + secondary ghost 1종으로 단순화
- 섹션 제목: `section-head`를 모든 주요 페이지에서 공통 사용

## 5) 아이콘/이모지 원칙
- 장식용 이모지는 최소화하고, 섹션 진행 순서를 나타내는 **번호형 표기(01~04)** 우선.
- 아이콘은 의미 보조 역할로만 사용하며, 제목 텍스트보다 시각 우선순위를 갖지 않게 유지.

## 6) 리듬/간격 원칙
- 페이지 그룹 간격은 `section-gap`으로 통일.
- 카드 내부는 `space-y-4~5`, 섹션 단위는 `space-y-3~4`로 계층 유지.
- 본문-보조텍스트-액션 순서가 반복되도록 구조화해 "읽히는 리포트" 인상을 강화.

## 7) 피해야 할 시각 요소
- 과도한 금색/네이비+보라 신비주의 조합
- 별자리/우주 장식 중심의 점술풍 패턴
- 과한 글로우, 과장된 애니메이션, 강한 다중 그림자
- 섹션마다 전혀 다른 톤을 사용하는 파편화된 UI
