# 시각 언어 가이드 (컬러 시스템)

## 1) 기본 팔레트
- 페이지 배경: `#FCF8F7`
- 소프트 강조 배경: `#F6E8EC`
- 카드 배경: `#FFFFFF`
- 기본 텍스트: `#000000`
- 보조 텍스트: `#666666`
- 약한 보조 텍스트/placeholder: `#999999`
- 메인 버튼: 배경 `#000000`, 텍스트 `#FFFFFF`
- 기본 보더: `#E9E9E9`

## 2) 사용 원칙
- 텍스트는 무채색만 사용한다.
- 강조는 색 추가 대신 카드 구조, 타이포 굵기/크기, 배경 톤 차이로 해결한다.
- 홈 /match /result/[id]에서 동일한 버튼/카드/보더 규칙을 유지한다.
- 보라/분홍/자주 계열 메인 인상과 과한 그라디언트는 사용하지 않는다.
