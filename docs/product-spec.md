# Product Spec (MVP 1차)

## 1) 서비스 목표
- 사주 기반 궁합 서비스의 핵심 사용자 여정(홈 → 입력 → 결과)을 빠르게 검증한다.
- 과도한 기능 구현 없이, 안정적인 화면/라우트/문서 골격을 먼저 구축한다.

## 2) 이번 단계 범위
- Next.js App Router 기반 필수 페이지 구성
- 공통 레이아웃 및 placeholder 컴포넌트 정리
- 정책/안내 문서 초안 반영
- 실행 가능 상태(dev/build) 확인

## 3) 다음 단계에서 할 일
- 입력 폼 상세 설계 및 유효성 검증
- 실제 궁합 계산 로직 연결
- 결과 해석 템플릿 고도화
- room/join 협업 플로우 기획 및 구현
- 분석 이벤트/로그 수집 구조 도입
