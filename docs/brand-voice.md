# 브랜드 톤앤매너 가이드 (1단계)

## 0) 현재 톤의 문제 요약
- 화면별 카피 결은 부드럽지만, 홈·입력·결과에서 문장 온도와 CTA 어조가 조금씩 달라 하나의 브랜드처럼 느껴지는 힘이 약했다.
- 일부 표현이 "궁합 서비스 일반 문구"에 가까워, 우리 서비스만의 인상(정리된 관계 리포트 경험)이 충분히 선명하지 않았다.
- 안내/검증/공유 상태 문구가 기능 안내 중심으로 흩어져 있어, 같은 브랜드가 말하는 느낌으로 묶이지 않았다.

## 1) 브랜드 한 줄 정의
**사주 정보를 바탕으로 관계의 흐름을 차분하게 읽어 주는, 부드럽고 정리된 프리미엄 관계 리포트.**

## 2) 브랜드 키워드 5개
1. 차분한 해석
2. 따뜻한 정리
3. 현실적 관찰
4. 선명한 구조
5. 가벼운 실천

## 3) 브랜드가 주는 감정
- "판단받는다"보다 **이해받는다**는 감정
- "맞다/틀리다"보다 **함께 읽어 본다**는 감정
- "자극적 재미"보다 **정리된 안심감**
- "무거운 상담"보다 **일상에서 바로 쓰는 명료함**

## 4) 추천 말투
- 20~30대 모바일 사용자가 자연스럽게 읽는 짧은 문장
- 단정 대신 관찰형 서술: "~일 수 있어요", "~를 살펴보면 좋아요"
- 감성은 유지하되 과장하지 않는 어휘: "흐름", "호흡", "조율", "정리"
- 결과 강조보다 활용 강조: "읽고 끝"이 아닌 "관계 대화에 써먹는 리포트"

### 문장 샘플
- "두 사람의 흐름을 한눈에 정리한 관계 리포트를 준비했어요."
- "확정적인 예측보다, 지금 관계를 읽는 참고 해석에 가깝습니다."

## 5) 피해야 할 말투
- 예언/단정형: "반드시", "운명", "100%", "정해진 미래"
- 과한 신비주의: "소름 돋는", "충격", "절대 피할 수 없는"
- 과도하게 무거운 상담체: "치유", "처방", "진단"
- 유치한 과잉 친근체/밈톤: "완전 찰떡", "레전드 궁합"
- 경쟁 서비스 핵심 분류어·구조를 연상시키는 반복 표현

## 6) CTA 말투 원칙
- 행동 + 결과를 함께 제시한다.
  - 예: "리포트 작성 시작", "관계 리포트 확인하기"
- 조급함 유도 대신 준비감을 준다.
  - 예: "입력 정보를 바탕으로 리포트를 정리해요"
- 버튼/보조 문구의 어조를 맞춘다.
  - 버튼은 짧게, 보조 문구는 안심형으로

## 7) 결과 리포트 말투 원칙
- 상단: 점수보다 **해석 맥락** 먼저 제시
- 섹션명: 강점/조율/실천처럼 **관계 활용 언어** 사용
- 문장: "예측"보다 "관찰" 중심
- 안내: "참고용 해석"을 명확히 고지

## 8) validation / empty state / 공유 문구 원칙
- Validation: 지적보다 안내
  - "오류" 대신 "이 정보를 먼저 채워 주세요"
- Empty state: 실패감보다 재시도 동선 강조
  - "찾을 수 없음" + "왜 그럴 수 있는지" + "바로 할 수 있는 다음 행동"
- 공유 문구: 자랑형보다 동행형
  - "같은 리포트를 함께 읽어 보세요"

## 9) 경쟁 서비스와 차별화되는 표현 방향
- "점술 상품" 인상 대신 **관계 리포트 경험**을 전면화
- 카피 구조를 "흥미 유도"보다 "해석 → 정리 → 활용" 순으로 고정
- 금지 표현(예: 모임 궁합/나와 궁합/둘만의 궁합/나만의 사주) 대신
  - "관계 리포트", "두 사람 흐름 해석", "조율 포인트"로 통일
- 관계를 "이어준다"는 추상 정서보다, **이미 있는 관계를 더 선명하게 읽어 주는 서비스**로 포지셔닝

## 10) 화면 적용 체크리스트 (홈 /match /result/[id])
- [x] 홈: 소개/CTA/준비중 문구를 리포트 경험 중심으로 통일
- [x] /match: 입력 안내·도움말·검증 문구를 "리포트 준비" 톤으로 통일
- [x] /result/[id]: 요약·섹션 설명·공유·빈 상태 문구를 같은 브랜드 어조로 정리
